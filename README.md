# SistemaSeguridadObserver
Proyecto en Java que implementa el Patrón de Diseño Observador en un contexto de seguridad doméstica. Permite simular sensores que detectan eventos (movimiento, gas, incendio, etc.) y notificar automáticamente a diferentes dispositivos conectados, como alarmas, cámaras de seguridad, ventilación automática y notificaciones móviles.
